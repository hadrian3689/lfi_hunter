# Project Title

LFI Hunter

## Description

A Local File Inclusion finder.

## Getting Started

### Executing program

* With python3
```
python3 lfi_hunter.py -u 'http://lfi.location/example.php?parameter=' -p '../../../../../' -w list.txt -o output.txt
```

## Help

For help menu:
```
python3 lfi_hunter.py -h
```
